BST Time and Space Complexity Analysis

The performance of Binary Search Tree operations varies significantly between average and worst cases, depending on the tree's balance.

Operation Average Case Worst Case Space When Balanced
Search O(log n) O(n) O(1) O(log n)
Insert O(log n) O(n) O(1) O(log n)
Delete O(log n) O(n) O(1) O(log n)
Traversal O(n) O(n) O(h)* O(n)
Find Min/Max O(log n) O(n) O(1) O(log n)

Important Notes

  • * Space complexity for traversal refers to the recursion stack depth (h = height of tree)
  • Worst case occurs in a skewed tree (essentially a linked list)
  • Average case assumes random insertions maintaining relative balance
  • Balanced trees (AVL, Red-Black) guarantee O(log n) operations
