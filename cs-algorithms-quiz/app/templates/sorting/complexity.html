Algorithm Complexity Analysis

Algorithm Time (Average) Time (Worst) Space Stable Best Use Case
Bubble Sort O(n²) O(n²) O(1) Yes Small datasets, nearly sorted data
Quick Sort O(n log n) O(n²) O(log n) No Large datasets, general purpose
Merge Sort O(n log n) O(n log n) O(n) Yes When stability is required
Heap Sort O(n log n) O(n log n) O(1) No Limited memory, guaranteed performance

Important Notes

  • Time complexity is measured in terms of comparisons and swaps
  • Space complexity refers to extra space needed beyond input array
  • Stability indicates whether equal elements maintain their relative order
  • Best use case considers practical factors beyond theoretical complexity
